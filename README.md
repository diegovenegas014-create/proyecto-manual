# Manual de Buenas Prácticas para Trabajos Universitarios

¡Bienvenidos al repositorio oficial de nuestro manual! 

## 📝 Descripción del Proyecto
Este proyecto recopila las mejores prácticas para realizar trabajos universitarios en grupo: organización, gestión de tiempos, formatos, citación y presentación. 

## 📂 Estructura del Repositorio
* `/documentos`: Borradores y versiones finales del manual.
* `/imagenes`: Recursos gráficos y capturas.
* `/referencias`: Material de apoyo y bibliografía.
